## Description
Freestyler = [FreestylerCore](https://github.com/cayugasoft/FreestylerCore) + a big set of styles.

This library is currently in development.
## Requirements
iOS 8.0+, Swift 2.3

## Installation
Could be installed via CocoaPods:
```ruby
pod "Freestyler", "~>0.1.0"
```
